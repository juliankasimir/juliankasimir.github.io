# Julian Kasimir – Persönliche Website

[![Built with Astro](https://astro.badg.es/v2/built-with-astro/small.svg)](https://astro.build)

Persönliche Website und Portfolio von Julian Kasimir mit Projekten, Ideen und Experimenten.

## Installation & Entwicklung

```bash
npm install        # Dependencies installieren
npm run dev        # Dev-Server starten auf localhost:4321
npm run build      # Build für Production
npm run preview    # Production-Build lokal anschauen
```

## Verwendete Technologie

Dieses Projekt basiert auf dem [Accessible Astro Starter](https://github.com/incluud/accessible-astro-starter) von [Incluud](https://github.com/incluud).

Die Komponenten stammen aus der [Accessible Astro Components](https://github.com/incluud/accessible-astro-components) Bibliothek.
